Projeto desenvolvido como estudo prático do **Curso em Vídeo**, no **Capítulo 17**, com foco na construção de uma página web estática utilizando **HTML5** e **CSS3**.

A proposta do projeto é apresentar uma página informativa sobre a **história do mascote do Android**, trabalhando estrutura semântica, estilização personalizada, imagens responsivas, uso de fontes externas e organização visual de conteúdo.

## Sobre o projeto

O `projeto-android` é uma página web simples, educativa e responsiva criada durante os estudos de HTML e CSS do Curso em Vídeo.

A página simula um artigo de tecnologia chamado **"História do Mascote do Android"**, trazendo curiosidades sobre o surgimento do Bugdroid, o mascote do Android, além de apresentar imagens, links externos, vídeo incorporado e uma seção complementar com versões do sistema Android nomeadas em homenagem a doces.

## Principais recursos

- Página estática feita com **HTML5** e **CSS3**;
- Estrutura de artigo com cabeçalho, menu, conteúdo principal, seção complementar e rodapé;
- Layout centralizado e responsivo;
- Imagens adaptáveis para diferentes tamanhos de tela;
- Uso da fonte personalizada `idroid.otf`;
- Uso de fonte externa via Google Fonts;
- Vídeo incorporado do YouTube;
- Links externos com abertura em nova aba;
- Lista com abreviações explicativas usando a tag `<abbr>`;
- Estilização com variáveis CSS;
- Organização de arquivos em pastas separadas.

## Como visualizar o projeto

Você pode abrir o projeto de forma simples no navegador.

### Opção 1: Baixar o repositório

Clone o projeto:

```bash
git clone https://github.com/thamiscoder/projeto-android.git
```

Entre na pasta:

```bash
cd projeto-android
```

Abra o arquivo:

```text
index.html
```

no navegador de sua preferência.

### Opção 2: Baixar ZIP

1. Acesse o repositório no GitHub;
2. Clique em **Code**;
3. Clique em **Download ZIP**;
4. Extraia os arquivos;
5. Abra o `index.html` no navegador.
